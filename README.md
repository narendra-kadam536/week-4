# Student Performance Analysis

## Project Description
This project analyzes student academic performance using Python, Pandas, and Matplotlib.

## Features
- Data cleaning and validation
- Subject-wise and student-wise analysis
- Data visualization
- Insightful reporting

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies using:
   pip install -r requirements.txt
2. Run the program:
   python main.py

## Output
- Charts saved in visualizations folder
- Analysis report in report folder
